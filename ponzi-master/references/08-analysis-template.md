## 一、项目分析五步法

### Step 1: 识别盘型

**目标**：判断项目属于哪种盘或组合

**分析维度**：

| 问题 | 分红盘 | 互助盘 | 拆分盘 |
|------|--------|--------|--------|
| 是否形成资金池？ | ✅ 是 | ❌ 否 | ✅ 是 |
| 是否允许自由进出？ | ❌ 否 | ❌ 多数否 | ✅ 是 |
| 是否承诺固定收益？ | ❌ 否 | ✅ 是 | ❌ 否 |
| 收益来源 | 时间线性分红 | 资金期限错配 | 资产增值 |
| 用户沉没成本 | 高（矿机/质押） | 中（规则约束） | 低（可自由退出） |

**判断输出**：
- [ ] 纯分红盘
- [ ] 纯互助盘
- [ ] 纯拆分盘
- [ ] 分红+互助组合
- [ ] 分红+拆分组合
- [ ] 互助+拆分组合
- [ ] 三盘结合

### Step 3: 计算庄家利润来源

**目标**：找到抽水/套利空间

#### 分红盘庄家利润

```
profit = 总入金 - 总固定分红
```

**分析点**：
- [ ] 入金渠道（矿机销售、质押入口）
- [ ] 分红来源（币本位vs金本位）
- [ ] 矿商/矿池角色
- [ ] 做市策略

#### 互助盘庄家利润

```
profit = 总互助状态中资金 × %老鼠仓账户比例
```

**分析点**：
- [ ] 抽水机制（显性/隐性）
- [ ] 清算机制
- [ ] 回购/做市资金去向
- [ ] 老鼠仓布局

#### 拆分盘庄家利润

```
profit = 项目方抛售总利润
```

**分析点**：
- [ ] 筹码分布
- [ ] 拆分节奏控制
- [ ] 抛售时机
- [ ] 叙事控制

## 三、快速分析清单

### 3分钟快速判断

- [ ] 1. 这是什么盘型？

### 红旗警告

以下情况应该警惕：

- [ ] 承诺金本位固定收益（互助盘高风险）
- [ ] 无明确清算机制
- [ ] 出金规则模糊
- [ ] 拆分速度过快
- [ ] 庄家利润来源不透明
- [ ] 与当前市场Beta严重不匹配
- [ ] 流动性来源不清晰

## 四、常见项目类型快速参考

### 矿币/DePIN

**盘型**：分红盘

**关键指标**：
- 矿机销量
- 产出流通量
- 筹码集中度
- 做市安排

**崩盘信号**：
- 矿机销量下滑
- 筹码分散
- 无上所计划

### 流动性挖矿/Yield Farming

**盘型**：互助盘 + 分红盘特征

**关键指标**：
- TVL vs 全局债务
- APY可持续性
- 清算机制

**崩盘信号**：
- APY下降
- TVL流失
- 踩踏风险

### NFT/铭文/Meme

**盘型**：拆分盘

**关键指标**：
- 拆分速度（新标的涌现）
- 存量持仓意愿
- 市场Beta

**崩盘信号**：
- 拆分饱和
- 存量抛售
- Beta被碾压

### 公链生态

**盘型**：三盘结合

**关键指标**：
- 质押锁仓（分红盘）
- 生态TVL（互助盘）
- 新项目数量（拆分盘）

**崩盘信号**：
- 质押解锁潮
- TVL流失
- 生态活跃度下降

### Restaking类

**盘型**：三盘结合（互助盘主导）

**关键指标**：
- TVL规模
- 下游协议数量
- APY可持续性

**崩盘信号**：
- 杠杆风险暴露
- 下游协议崩溃
- ETH价格大跌

```

```

### 示例

```
MoneyArk 是一个 互助盘带分红盘特征 ，庄家通过
85%回购资金的波段操作+老鼠仓 赚钱，
当 全局债务超过MARK底池可清算资产 时会崩盘，
当前时机 不匹配（高Beta牛市环境） ，
建议 观望或在TVL与LP失衡时果断撤出 。
```
