# Tweet Templates & Examples

## Thread Templates

### Knowledge Thread (教学型)

**EN:**
```
🧵 [Number] [topic] lessons I learned [context]:

1/ [Bold opening insight]

[2-3 sentences expanding with specific example]

2/ [Second insight]

[Supporting evidence or anecdote]

...

[N]/ Summary:

→ [Key point 1]
→ [Key point 2]
→ [Key point 3]

If this was useful, repost ♻️ so others can find it.

Follow @[handle] for more on [topic].
```

**CN:**
```
🧵 关于[主题]，我总结了[数字]条经验：

1/ [开头金句]

[2-3句展开，附具体案例]

2/ [第二条洞察]

[数据或故事支撑]

...

[N]/ 总结一下：

→ [要点1]
→ [要点2]
→ [要点3]

觉得有用的话转发让更多人看到 ♻️
```

### Breakdown Thread (拆解型)

```
I spent [time] studying [subject].

Here's what 99% of people get wrong:

🧵👇

1/ [Common misconception]

The truth: [counter-intuitive insight]

...
```

### Story Thread (故事型)

```
[Time] ago, I [situation].

[Unexpected outcome].

Here's exactly what happened (and what you can learn from it):

🧵

1/ [Set the scene]
...
```

## Single Tweet Templates

### Insight Tweet
```
[Observation about domain].

Most people think [common belief].

But [counter-intuitive truth].

Here's why: [1-2 sentence explanation]
```

### Framework Tweet
```
The [Name] Framework for [outcome]:

1. [Step 1] → [result]
2. [Step 2] → [result]
3. [Step 3] → [result]

[One-line takeaway]
```

### List Tweet
```
[N] [things] that [benefit]:

• [Item 1]
• [Item 2]
• [Item 3]
...

Which one resonates most? 👇
```

### Hot Take Tweet
```
Unpopular opinion:

[Bold statement]

Here's why:

[2-3 lines of reasoning]
```

### Bilingual Insight (双语金句)
```
[English insight in 1-2 lines]

[中文翻译或延伸思考]

——[attribution or context]
```

## Engagement Tweets

### Poll-style
```
[Question about audience preference or belief]

Option A: [choice]
Option B: [choice]
Option C: [choice]

Reply with yours 👇
```

### Question Tweet
```
Genuine question for [audience]:

[Specific, answerable question]

I'll share my take tomorrow based on your responses.
```

## Content Calendar Weekly Structure

| Day | Theme | Format | Language |
|-----|-------|--------|----------|
| Mon | Industry insight / trend | Thread (5-8 tweets) | EN |
| Tue | Product thinking / framework | Single + engagement | CN |
| Wed | Technical deep-dive | Thread (8-12 tweets) | EN |
| Thu | Personal experience / story | Story thread or single | CN |
| Fri | Curated resources / tools | List tweet | Bilingual |
| Sat | Hot take / opinion | Single tweet | Flexible |
| Sun | Reflection / week recap | Thread (3-5 tweets) | Flexible |
