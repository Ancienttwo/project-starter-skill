## Task Management Protocol

```yaml
TASK_SOURCES:
  - tasks/todo.md
  - tasks/lessons.md
  - docs/PROGRESS.md  # compatibility history

RULES:
  - Plan first in tasks/todo.md with checkable items
  - Track progress with evidence
  - Keep a REVIEW_SECTION in tasks/todo.md
  - Record correction-derived prevention rules in tasks/lessons.md

COMPATIBILITY:
  - docs/plan.md stores deep architecture notes
```
