# Workflow Orchestration (Reference)

Use this file for advanced orchestration and planning patterns.

## Default Loop

1. Plan in `tasks/todo.md`.
2. Execute in small verified slices.
3. Record evidence in review section.
4. Capture prevention rules in `tasks/lessons.md`.

## Multi-Track Execution

- Split only independent tracks.
- Assign one owner per track.
- Merge after each track has verification proof.

## Completion Gate

A task is complete only when behavior is verified and risks are documented.
